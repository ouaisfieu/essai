# La Ligne — Dossier citoyen sur la rupture du contrat social belge (1944–2026)

Un dossier web réflexif, écrit du point de vue du citoyen, sur le démantèlement de
l'État social belge sous la coalition Arizona (2024–2026) : le pacte de 1944, les
réformes de 2025-2026, l'« effet de falaise » du statut BIM (avec simulateur
interactif), la contagion sociale, la bataille constitutionnelle autour de
l'article 23 et du principe de *standstill*, et la mobilisation de la rue.

## Contenu
- `index.html` — le dossier complet, en pur HTML/CSS/JS, sans dépendance ni traceur
  (seul appel externe : Google Fonts).

## Publication
Le fichier est autonome. Pour le publier avec GitHub Pages :
Settings → Pages → Deploy from a branch → `main` / racine.

## Méthode et signature
Synthèse rédigée et mise en page par **Claude** (Anthropic), modèle d'intelligence
artificielle, à partir d'un corpus de cinq études documentaires réuni par le
porteur du projet (juillet 2026). Le dossier assume un point de vue citoyen tout
en présentant les trois registres du débat (militant, analytique, gouvernemental).
Certains chiffres sont des données précoces et contestées ; les corrections
documentées sont bienvenues via les issues du dépôt.
